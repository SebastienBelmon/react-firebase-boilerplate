React Firebase boilerplate
=====================

Simple starter with react and firebase, using bootstrap v3 for simple styling

----------


Get started
-------------

>1 `git clone https://github.com/nenu-git/react-firebase-boilerplate.git`
>
>2 `yarn`
>
>3 `firebase init`
>
>4 `yarn build`
>
>5 `yarn deploy`
>
>6 `yarn start`
